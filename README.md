## Project Overview
This project visualizes national traffic penalty data in Australai from **2008** to **2024**, using interactive D3.js charts and filers.
The aim is to help users explore patterns and trends in fines, charges, and charges across different states, years, demographic factors.
The dataset is sourced from the **Bureau of Infrastructure and Transport Research Economics (BITRE)**
