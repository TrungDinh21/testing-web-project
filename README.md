## Project Overview
This project visualizes national traffic penalty data in Australai from **2008** to **2024**, using interactive D3.js charts and filers.
The aim is to help users explore patterns and trends in fines, charges, and charges across different states, years, demographic factors.
The dataset is sourced from the **Bureau of Infrastructure and Transport Research Economics (BITRE)**

## StoryBoard
The project website is structured into five main pages that guide users through different stages of exploring Australia’s road safety enforcement data. Each page has a distinct purpose, contributing to the overall narrative of fairness and transparency in enforcement across jurisdictions.

1. Home Page

Introduces the project background, motivation, and objectives.
It explains the purpose of analysing enforcement data from 2008–2024 and how the visualizations promote transparency and understanding of road safety policies.

2. National Overview Page

Provides a nationwide perspective on enforcement trends.
Features an interactive map of Australia showing penalty distribution across eight jurisdictions and a line chart illustrating national trends from 2008 to 2024.

3. State Trends Page

Allows comparison of penalty trends between jurisdictions.
Includes a multi-line chart showing total penalties by state and a histogram displaying penalty distribution, helping users identify how enforcement intensity changes over time.

4. Fairness Comparison Page

Focuses on evaluating fairness in enforcement.
Displays two bar charts — one showing total penalties and another showing penalties per 10,000 driver licences — allowing population-adjusted comparisons across states.

5. About Page

Provides information about the project group, including group name, member details, and sections on data governance, AI declaration, and project limitations, ensuring transparency and ethical compliance.
