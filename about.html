About the Data

Privacy

  • The datasets used in this project contain only aggregated enforcement figures at state level.
  • No personal, private, or individual identifiable information is included.
  • All data compiles with Australian Governement open-data privacy standard.

Accuracy & Limitations

The available data has varying degrees of completeness across years:

  • The datasets used in this project contain only aggregated enforcement figures at state level.
  • No personal, private, or individual identifiable information is included.
  • All data compiles with Australian Governement open-data privacy standard.

Therefore, long-term comparisons primaryli focus on fines.

Bias

Raw penalty counts can be influenced by differences in:

  • The datasets used in this project contain only aggregated enforcement figures at state level.
  • No personal, private, or individual identifiable information is included.
  • All data compiles with Australian Governement open-data privacy standard.

To reduce bias, the project includes penalties per 10,000 driver licences for a fairer interstate comparion.
These results should be interpreted as indicative, not absolute.

Data Ethics

  • All datasets are openly available public government records.
  • Used solely for educational and analytical purposes.
  • No individual-level data accessed or handled.
  • Visualizations aim to support transparency, not target or judge states or demographics.
